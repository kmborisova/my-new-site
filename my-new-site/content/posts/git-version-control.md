# Что такое Git и зачем он нужен?

Git — это система контроля версий, которая помогает отслеживать изменения в коде и работать в команде.

## Основные команды Git

* `git init` — создать новый репозиторий
* `git clone` — скопировать удалённый репозиторий
* `git add` — добавить изменения в индекс
* `git commit` — сохранить изменения в истории
* `git push` — отправить изменения на GitHub
* `git pull` — скачать изменения с GitHub

## Почему Git важен?

Git позволяет:
- Откатываться к старым версиям кода
- Работать над разными функциями в отдельных ветках
- Сотрудничать с другими разработчиками
- Хранить код в надёжном месте (например, на GitHub)
